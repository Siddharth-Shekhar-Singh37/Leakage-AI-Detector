This project is currently under active development.

## 📌 Overview
An end-to-end intelligent system that:
- Detects water leakage anomalies from MNF, pressure and acoustic data
- Uses AI (Claude API + LangChain) to generate plain-English reports
- Automates report delivery via email and dashboard
- Built with Python, Streamlit, SQLite, dbt and n8n

## 🛠️ Tech Stack
`Python` `Pandas` `SQLite` `dbt` `LangChain` `Claude API` `Streamlit` `n8n`
